# Changelog

## [0.5.0] - 2026-03-26

### plugin-social-monitor (NOVO)
- Monitoramento de menções nas redes sociais (Twitter, Instagram, TikTok, Reddit, LinkedIn)
- Filtros por rede, sentimento (positivo/neutro/negativo) e categoria (Jusbrasil, Jus IA, Experience)
- Termos monitorados editáveis pelo usuário
- Dashboard widget com menções recentes
- Preparado para integração Apify (dados mock por enquanto)

### plugin-creative-hub
- **Galeria com 3 abas:** Aguardando / Reprovados / Aprovadas
- **Descartar definitivamente:** checkbox no modal de reprovação, arquiva sem feedback
- **Remoção instantânea:** ao aprovar/reprovar/descartar, item some sem reload de página
- **Briefing form:** botão "Novo briefing" na Timeline com formulário completo
- **Calendário editorial:** página /calendario com visão semanal
- **Métricas de produtividade:** KPIs, produção semanal, por editoria, por agente
- **Galeria mostra todas as issues** com imagens (limit 200, sem slice)
- Calendário removido da sidebar (página acessível via URL direta)

### plugin-content-review
- Toolbar integrada sem botão Back
